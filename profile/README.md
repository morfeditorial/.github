## About Us

Morf OSS is an open-source organization focused on building modern software for content publishing, digital communities, and creators. We develop modular applications, reusable libraries, and developer tools with an emphasis on clean architecture, extensibility, and long-term maintainability.

## Our Work

- Developing **Machinima**, a modular Symfony-based platform for publishing machinima content and building creator communities.
- Building reusable PHP libraries and Symfony bundles designed for modern application development.
- Designing platform-agnostic architectures based on contracts, adapters, and dependency injection.
- Publishing open-source tools, libraries, and infrastructure for the PHP ecosystem.
- Maintaining documentation, examples, and developer resources for our projects.
